Hi

My name is Aditi. I hold a BS in computer science + a minor in studio art from UVA and am currently a MS student at NYU studying integrated design and media.

How do you like my website?

I've thought a lot about the Handmade Web and how special it is to create websites that are rich in character and have room for change. I like to think of web dev as a practice of crafting and not this finalized and pristine thing that will stay the same forever.

I've set up the main page to be a playground of some of my projects (serious and fun). Some are more code heavy experiments while others are more artistic. I've organized the project index on a scale of code to art so you can get a feel of how they mesh together in my brain.
